# New Observation

Create a new field observation with the following steps:

1. Run: python practice_project/scripts/create_observation.py
2. Run: python practice_project/scripts/create_metadata.py
3. Run: python practice_project/scripts/create_notes.py
4. Run: python practice_project/scripts/log_observation.py
5. Run: python practice_project/scripts/check_metadata.py

Then report which files were created and confirm the observation was logged in observations.csv.
